## Claude logo Animation
本项目是 对 Anthropic 官网 [When AI builds itself](https://www.anthropic.com/institute/recursive-self-improvement) 文章中 Hero部分 中的动画的约 80% 复刻。
由于是个人通过简单'逆向' 官网 的 Next.js 打包后的 js 代码并提交给 Gemini 复刻，所以本项目不通过 prompt.md，如需参考实现可看[源码](./index.html)或者这篇[文章](https://anningui.github.io/posts/claude-logo-animation/)。
